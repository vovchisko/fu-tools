# fu-tools

I just wanted to speedup load time for my tiny projects.

> Some functions behave a bit differently from classic lodash.
> I warned you.

### Install from github using `npm`:

```bash 
npm i fu-tools
```

### List of functions

- `isPlainObject`
- `isArray`
- `isString`
- `isBoolean`
- `isNumber`
- `isNaN`
- `isNull`
- `isSet`
- `isMap`
- `isActualNumber`
- `cloneDeep`
- `isEqual`
- `get`
- `set`
- `flatten`
- `omit`
- `pick`
